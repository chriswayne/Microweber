PHP Code Cleaner 0.7
By : Sabeen Malik
www.decodephp.com



Place the contents of this directory into your web folder. Set permissions of folder "output" to 0777.

Place the files you want to clean in "clearthese" folder and run index2.php, clean files will be placed in the "output" folder.

To clean the code manually run index.php via your web browser and paste the code in the first box to clean.
